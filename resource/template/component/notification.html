通知方式说明

配置字段

URL通知入口地址，可包含消息占位符
#NODEKEEP#告警消息占位符，可用于 URL 和 Body
GET只拼接 URL，不发送 Body
POST发送 Body，并根据请求类型设置 Content-Type
JSONBody 按 application/json 发送
FORMBody 按 application/x-www-form-urlencoded 发送

占位符处理

URL占位符会进行 URL 编码 JSON占位符会按 JSON 字符串转义 FORMBody 需要填写 JSON 对象，值内可使用占位符 VerifySSL启用后校验 HTTPS 证书

示例

{
  "title": "nodekeep",
  "content": "#NODEKEEP#"
}

如果通知服务要求表单提交，请选择 FORM，并把 Body 写成 JSON 对象，例如 {"msg":"#NODEKEEP#"}。
