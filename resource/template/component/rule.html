告警规则说明

规则字段

type指标类型
min低于此值触发，0 表示不限制
max高于此值触发，0 表示不限制
duration持续时间，至少 3 秒
ignore忽略指定服务器 ID

指标类型

offline节点离线 cpuCPU 使用率 memory内存使用率 swapSwap 使用率 disk磁盘使用率 net_in_speed入站网速 net_out_speed出站网速 net_all_speed总网速 transfer_in累计入站流量 transfer_out累计出站流量 transfer_all累计总流量

示例

[
  {"type":"offline","duration":30},
  {"type":"cpu","max":90,"duration":60},
  {"type":"memory","max":90,"duration":60},
  {"type":"disk","max":85,"duration":60,"ignore":{"1":true}}
]

同一告警规则内的子规则全部持续异常时才会触发；当前不支持按服务器分组指定规则。
